# Three.js

[Three.js](https://threejs.org/)는 웹 브라우저에서 3차원 컴퓨터 그래픽스 애니메이션 애플리케이션을 만들고 표현하기 위해 사용되는
자바스크립트 라이브러리이자 API이다. 오픈 소스 프로젝트로 깃허브에서 공개되어 있다. ([위키백과](https://ko.wikipedia.org/wiki/Three.js))

React 환경에서는 [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)와
[Drei](https://github.com/pmndrs/drei#readme) 라이브러리를 통해 보다 편리하게 Three.js를 연동할 수 있다.
